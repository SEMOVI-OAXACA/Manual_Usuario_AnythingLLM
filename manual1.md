# Manual de operación para AnythingLLM

## Acceso al Sistema
Para iniciar la interacción con el sistema AnythingLLM, siga los siguientes pasos:

1. Acceso a la Plataforma: Ingrese a la siguiente dirección web:
![Pagina inicio](/img/2.PNG)
``
https://soporte.apps.semovioaxaca.gob.mx/
``

2. Credenciales de Acceso: Introduzca el nombre de usuario y la contraseña que le fueron asignados.
![Inserte usuario y contraseña](/img/3.PNG)

## Gestión y Realización de Consultas
Siga estas indicaciones para interactuar con el asistente de soporte:

1. Inicio de una Nueva Consulta: Para comenzar una nueva sesión de preguntas o peticiones, diríjase al panel de control y seleccione la opción "New Thread". Esto garantiza una consulta limpia y sin interferencias de conversaciones previas.
![Abrir hilo](/img/4.PNG)

2. Formulación de la Consulta: Redacte su pregunta o petición de la manera más clara, concisa y específica posible en el campo de texto correspondiente. Una formulación precisa optimiza la calidad de la respuesta del sistema.
![Hacer consulta](/img/5.PNG)

## Manejo de Respuestas Incompletas o Incorrectas:

- Primer Intento: Debido al amplio volumen de datos procesados por el modelo, ocasionalmente el sistema puede generar respuestas incompletas. Si esto ocurre, se recomienda volver a realizar la petición haciendo hincapié en la necesidad de que se proporcione la información completa.

- Fallo Persistente: Si la respuesta persiste incompleta, es incorrecta, o si el sistema no proporciona ninguna información, se debe:

    - Consultar la documentación oficial disponible para validar la información.

    - Notificar a la Unidad Informática inmediatamente sobre el fallo para su revisión y corrección.


## Recomendaciones Generales de Uso
Para garantizar un uso eficiente y adecuado de la herramienta, tenga en cuenta las siguientes directrices:

Alcance Temático: El sistema está destinado exclusivamente para consultas relacionadas con el soporte de trámites de SEMOVI Oaxaca. Evite realizar preguntas o peticiones ajenas a esta temática.

Claridad y Especificidad: Sea lo más específico y claro posible en la redacción de cada consulta.

Gestión de Hilos: Una vez que un hilo de conversación haya cumplido su propósito y ya no sea requerido, se recomienda eliminarlo para mantener la plataforma organizada y optimizada.
